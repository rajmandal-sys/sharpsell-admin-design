# Voice and Tone

SharpSell is a B2B sales enablement platform used by sales teams and administrators during work. Every piece of text should help the user complete their task faster.

---

## Principles

| Principle | What it means |
|---|---|
| **Clear over clever** | Use plain language. Avoid jargon, puns, or marketing speak in the product UI. |
| **Brief over verbose** | Say it in fewer words. If a label or message can be shorter without losing meaning, shorten it. |
| **Helpful over neutral** | Guide the user toward the right action. Don't just state facts — tell them what to do. |
| **Consistent over creative** | Use the same term for the same thing everywhere. Don't alternate between synonyms. |

---

## UI Copy Rules

### Labels and Headings
- Use sentence case: "Save changes", not "Save Changes"
- Keep labels under 3 words when possible
- Be specific: "Upload document" beats "Add file"

### Button Text
- Start with a verb: "Save", "Delete", "Upload", "Create"
- Describe the action, not the destination: "Save changes" not "Done"
- Destructive actions must be explicit: "Delete user" not "Remove"

### Error Messages
- Say what went wrong: "Email address is required"
- Say how to fix it: "Enter a valid email address"
- Never blame the user: "Invalid input" → "This field requires a number"

### Empty States
- Explain why the area is empty: "No users have been added yet"
- Offer the next step: "Add your first user to get started"

### Confirmation Dialogs
- State the consequence: "This will permanently delete 3 users"
- Use the destructive action as the button label: "Delete 3 users"
- Offer a safe exit: "Cancel"

## Terminology

Use these terms consistently across the platform:

| Use this | Not this |
|---|---|
| Save | Submit, Apply |
| Cancel | Dismiss, Close, Nevermind |
| Delete | Remove, Erase, Trash |
| Create | Add, New, Make |
| Search | Find, Look up |
| Upload | Attach, Import |
| User | Member, Account |
| Enable / Disable | Turn on / Turn off |

## Tone by Context

| Context | Tone | Example |
|---|---|---|
| Success | Confident, brief | "Changes saved" |
| Error | Direct, helpful | "Could not save — check your connection and try again" |
| Warning | Calm, clear | "This action cannot be undone" |
| Empty state | Encouraging | "No results yet — try adjusting your filters" |
| Loading | Transparent | "Loading users..." |
